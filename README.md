# TaiwanWeatherChartGenerator
抓取中央氣象局的開放資料（json），來輸出圖表。本例使用「臺北市未來1週天氣預報」來進行開發測試。

## 參閱文件
1. [交通部中央氣象局-開放資料平臺-資料使用說明](http://opendata.cwb.gov.tw/usages)
2. downloads/CWB_Opendata_API_V1.2.pdf
